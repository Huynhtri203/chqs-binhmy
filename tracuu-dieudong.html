HƯỚNG DẪN KẾT NỐI GOOGLE SHEETS

1
Hàng tiêu đề Sheet3 phải đúng thứ tự: so_quyet_dinh ho_ten cccd ngay_sinh don_vi_cu don_vi_moi chuc_vu ngay_quyet_dinh hieu_luc pdf_url ghi_chu
2
Publish to web: File → Share → Publish to web → chọn Sheet3 → CSV → Publish. Copy link CSV nhận được.
3
Điền link CSV vào biến CSV_URL trong <script> cuối file HTML này.
4
Upload PDF lên Google Drive → Get link → đổi /view thành /preview → điền vào cột pdf_url trong Sheets.
5
⚠️ Phải chạy qua http:// server (VS Code Live Server, Python http.server...) — không mở file:// trực tiếp vì trình duyệt chặn fetch().
